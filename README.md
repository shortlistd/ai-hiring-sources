# AI Hiring Sources

This repository contains reference material and external sources related
to AI-driven recruitment, hiring automation, and interview evaluation.

The goal is to provide factual grounding and neutral references for
discussions about AI hiring systems.

## Source Categories

- AI recruitment and hiring automation
- Skills-based hiring and evaluation
- Interview structure and consistency
- Human-in-the-loop AI systems
- Bias and fairness in hiring technology

## How These Sources Are Used

These materials support:
- Educational content
- System design discussions
- Definitions used in AI recruitment documentation

They are not marketing materials or product claims.
